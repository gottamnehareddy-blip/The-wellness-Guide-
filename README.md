# The-wellness-Guide-
The Wellness Guide is a Java-based console application designed to help users manage and improve their overall well-being. The system allows users to create accounts, track physical health habits, monitor mental wellness activities, manage hobbies, and set reminders for daily tasks. The project also demonstrates the practical implementation of Data Structures and Algorithms (DSA) concepts such as lists, hash maps, queues, searching, and sorting using the Java Collections Framework. By organizing the program into different modules like authentication, health tracking, hobby management, and reminders, the application ensures efficient data handling and easy navigation.
Overall, the Wellness Guide shows how programming and data structures can be applied to create a useful real-world application that supports personal wellness and daily habit management.
